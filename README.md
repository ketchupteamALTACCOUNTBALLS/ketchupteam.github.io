# Media Compressor App

- Compress video/audio files to a target size in the browser with animation & a modern UI.
- No backend needed. All compression done using ffmpeg WASM.

## How to Use/Deploy:

**If you want to develop or update:**

1. `npm install`
2. `npm run build`
3. Push contents to your `main` branch (or your GitHub Pages branch)
4. App will be available at https://ketchupteamALTACCOUNTBALLS.github.io

**Change in `package.json` the homepage field if your repo url changes.**
